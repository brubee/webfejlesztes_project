About this website:

This website (and thus, this project) was created for the Web Development class of 2023/2024.
The goal of this project was to create a functional, responsive website that allows logged-in users to upload their opinions and ratings about movies they had seen. It was important that there are at least two sites that communicate with each other and a database, along with a proper security system (which here, means the log-in function).
